# 🧠 Sklearn Clone (From Scratch)

A lightweight, from-scratch implementation of core machine learning algorithms inspired by **scikit-learn** — built using pure Python (and optionally NumPy).

This project is part of my journey to deeply understand ML internals by recreating them step-by-step.

---

## 🚀 Features

### 📊 Linear Models
- Linear Regression
- Logistic Regression

### 🔧 Preprocessing
- MinMaxScaler
- StandardScaler

### 📏 Metrics
- R² Score
- Accuracy
- (More coming soon...)

```bash

cd sklearn-clone
```

- `requiements install`  

``` bash
pip install -r requirements.txt
```
